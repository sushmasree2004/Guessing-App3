# Guessing-App

A simple Java console-based number guessing game where the player tries to guess a randomly generated number within a limited number of attempts. The game provides hints after wrong guesses to make it more engaging.


Features

Random number generation between 1 and 100
Maximum of 7 attempts to guess correctly

Up to 3 hints provided:
Odd or Even
Greater than 50 or not

No more hints after 3 wrong guesses
Input validation with custom exception handling (InvalidInputException)
Clear feedback for guesses (LOW, HIGH, CORRECT)
